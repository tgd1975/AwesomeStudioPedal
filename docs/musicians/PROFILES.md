# Profiles

AwesomeStudioPedal ships with seven profiles. Press SELECT to cycle through them.
The three small LEDs show the active profile — see the [User Guide](USER_GUIDE.md) for the LED table.

---

## Profile 01 — Score Navigator

Use this when reading digital sheet music on a tablet or laptop (forScore, MobileSheets, Adobe Acrobat,
or any app that responds to Page Up / Page Down).

| Button | Name | What it does |
|--------|------|--------------|
| A | Prev Page | Go to previous page |
| B | Next Page | Go to next page |
| C | First Page | Jump to first page |
| D | Last Page | Jump to last page |

---

## Profile 02 — Pixel Camera Remote

Use this for hands-free video and photo control on a Google Pixel smartphone.

| Button | Name | What it does |
|--------|------|--------------|
| A | Delayed Shutter | Takes a photo after a 3-second countdown |
| B | Instant Shutter | Takes a photo immediately |
| C | Switch Mode | Switches between photo and video mode |
| D | Gallery Preview | Opens the gallery preview |

**Button A — Delayed Shutter:** Press A, then pick up your instrument. The photo triggers after
3 seconds. The power LED blinks while the countdown runs.

---

## Profile 03 — VLC Mobile Controller

Use this when practising along with a video or audio recording in the VLC app on your phone.

| Button | Name | What it does |
|--------|------|--------------|
| A | Play/Pause | Play or pause playback |
| B | Stop | Stop playback |
| C | Slow Down | Decrease playback speed |
| D | Speed Up | Increase playback speed |

---

## Profile 04 — OBS Stream Deck

Use this as a foot-operated scene switcher for OBS Studio on a PC or Mac.
Buttons send high-range function keys (F13–F16) that do not interfere with normal typing;
assign them to OBS hotkeys in OBS Settings → Hotkeys.

| Button | Name | What it does |
|--------|------|--------------|
| A | Intro Scene | Switch to intro scene (F13) |
| B | Main Scene | Switch to main scene (F14) |
| C | Mute Mic | Mute microphone (F15) |
| D | Start Stream | Start/stop stream (F16) |

---

## Profile 05 — DAW Looper (Ableton)

Use this for recording and looping in Ableton Live. Designed for the solo performer workflow,
where undoing a bad take quickly matters.

| Button | Name | What it does |
|--------|------|--------------|
| A | Record | Toggle record (F9) |
| B | Play/Stop | Play or stop (Space) |
| C | Undo Take | Undo last take (Ctrl+Z) |
| D | Metronome | Toggle metronome (C) |

---

## Profile 06 — Social and Comms

Use this for interacting with Discord, Slack, or Zoom while playing an instrument.

| Button | Name | What it does |
|--------|------|--------------|
| A | Quick Msg | Types: "Starting now, see you in the chat!" |
| B | Mute App | Mute in the active app (Ctrl+Shift+M) |
| C | Deafen | Deafen in the active app (Ctrl+Shift+D) |
| D | Screenshot | Take a screenshot (Print Screen) |

---

## Profile 07 — System Debug

This is a technical diagnostic profile, not intended for musical use.
Use it to verify the pedal is running and responding correctly.

| Button | Name | What it does |
|--------|------|--------------|
| A | Ping Pedal | Writes a status message to the serial monitor |
| B | Version Info | Types the firmware version string |
| C | Nav Up | Arrow Up |
| D | Nav Down | Arrow Down |
